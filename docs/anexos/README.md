# Anexos — Hito 5

Material de respaldo del proyecto. **Todo el contenido de esta carpeta ya está integrado dentro
del informe consolidado** (`docs/Hito_5_PowerStrike.docx` / `.pdf`); se conserva por separado como
fuente y evidencia.

Los entregables están un nivel arriba, en `docs/`:

- `Hito_5_PowerStrike.docx` / `Hito_5_PowerStrike.pdf` — informe completo (entregable).
